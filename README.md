# uuid_cli

A command line utility for genrating UUIDs

## Installation

```
cargo install uuid_cli
# or
git clone https://github.com/rakenodiax/uuid_cli.git && uuid_cli && cargo install --path .
```

## Usage

```
uuid
# prints 0a781908-9fad-4b4b-a558-d9f19513bfc2
```
